Types of views:

Possible actions:

*Chose bar by press - twice will open track on that bar

*Scroll gesture left\right - navigation

B) Track view

*Chose bar and beat by press

*Scroll tablature up\down 

*Double click sets end of the selection for coping

C) Tests view

*Contains files for alpha tests, first two sections are simple functions,
third - mine and mine friends compositions 

D) Tap rhythm vire

*Lets you to tap rhythm and make tab from it

E) Pattern input view

*Lets to create drums as pattern

F) Configutarion view

*Shows current configuration

*Scale buttons(zoom in and out)

*Logging buttons

*Gesture for pinch scale(works only on that window)

G) Record view

*Records audio
	
*Shows audio wave and lets navigate through it
	(not on android yet)

H) Morce Code Rhythm

<h2>Logging</h2>
<pre>
Also there is logging system to help find bugs or crash reasons.
Now log files could be found at the root of you sdcard,
log file name: Guitarmy.log

Please note, that every time you launch guitarmy - log gets replaced by new,
so if you need save log after crash - do it before launching Guitarmy again
</pre>
